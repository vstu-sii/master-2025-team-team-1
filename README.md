## 1. User Segments (Целевые сегменты пользователей)

- **HR-специалисты рекрутинговых агентств**  
  Отвечают за подбор большого количества кандидатов на разные позиции.  
  Им важно быстро отсеивать неподходящих и находить топ-кандидатов.

- **HR-отделы компаний среднего бизнеса**  
  Занимаются массовым подбором персонала.  
  Нужна автоматизация рутинных задач (парсинг резюме, сравнение с требованиями вакансий).

- **Руководители команд/менеджеры по найму**  
  Участвуют в интервью, но не тратят время на анализ резюме.  
  Им полезно получать персонализированные вопросы для собеседования.

## 2. Pain Points (Основные боли пользователей)

- Затраты времени на чтение и анализ десятков/сотен резюме.
- Высокий риск субъективной ошибки при отборе кандидатов.
- Сложность в выявлении реальных навыков и пробелов у кандидатов.
- Нет единой системы для парсинга резюме разных форматов (PDF, Word, текст).
- Недостаток инструментов для подготовки к интервью на основе данных кандидата.

## 3. Use-Cases (Ключевые сценарии использования)

- **Автоматическая фильтрация и ранжирование**  
  HR загружает пачку резюме и описание вакансии.  
  Система выдает скоринг соответствия и формирует рейтинг кандидатов.

- **Выявление ключевых навыков и пробелов**  
  Для выбранного кандидата AI показывает сильные стороны и зоны для развития.  
  HR получает объективную картину, снижая риск ошибки.

- **Подготовка интервью**  
  На основе опыта кандидата AI генерирует 5–7 персональных вопросов.  
  Менеджер тратит меньше времени на подготовку к собеседованию.

- **Always on display**  
  AI HR Assistant работает в качестве расширения для браузера.  
  Менеджер не тратит лишние действия на загрузку резюме кандидатов, а сразу может использовать приложение.

## 4. Value Proposition (Уникальная ценность продукта)

AI HR Assistant экономит время HR-специалистов и повышает качество найма, автоматически анализируя резюме, ранжируя кандидатов и подготавливая персонализированные вопросы для интервью.

## 5. North Star Metric (Главная метрика успеха)

- **Метрика:** Сокращение времени на первичный отбор кандидатов (скрининг).  
- **Базовый уровень:** ~60 минут на обработку 20 резюме вручную.  
- **Целевое значение:** ≤ 10 минут на обработку 20 резюме с помощью AI HR Assistant.
