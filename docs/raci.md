## Task Breakdown (Основные задачи проекта)
Ключевые области ответственности для проекта AI HR Assistant с учётом браузерного расширения:
1. Формулировка проблемы и целевой аудитории
2. Создание C4 архитектуры системы
3. Разработка roadmap и планирование спринтов
4. Определение North Star Metric
5. Написание ADR (Architecture Decision Records)
6. Контроль DoR/DoD (Definition of Ready/Done)
7. Проведение ретроспектив команды
8. Определение требований к моделям и метрикам
9. Создание и тестирование baseline моделей
10. Промпт-инжиниринг и оптимизация
11. Настройка RAG цепочек
12. Интеграция Langfuse для трассировки
13. A/B тестирование моделей
14. Анализ метрик: accuracy, latency, hallucination
15. Создание Docker Compose конфигураций
16. Настройка GitHub Actions CI/CD
17. Настройка Grafana/Prometheus
18. Автоматизация lint, test, build процессов
19. Управление контейнерами и окружениями
20. Мониторинг производительности системы
21. Разработка React/Next.js интерфейсов
22. Создание FastAPI backend сервисов
23. Проектирование UX-flow и use cases
24. Интеграция с bolt.ai/V0 для прототипов
25. Обработка ошибок и валидация
26. Responsive и адаптивная верстка
27. Интеграция с LLM через API
28. Разработка расширения для браузера

## Role Assignments (Назначение ролей)
- **SA/PO** — отвечает за продукт, приоритизацию задач, принятие решений.
- **AI Engineer** — отвечает за алгоритмы анализа резюме, скоринга и LLM.
- **MLOps** — отвечает за деплой моделей, мониторинг, CI/CD и инфраструктуру.
- **Fullstack** — отвечает за фронтенд, интеграции, браузерное расширение и backend.

## Decision Making Process (Процесс принятия решений)
- Все ключевые решения по приоритетам и функционалу принимает **SA/PO**.
- Обсуждения и предложения идей проходят на командных встречах.
- Конфликты разрешаются через согласование SA/PO.

## Escalation Path (Путь эскалации конфликтов)
1. Обсуждение и попытка разрешения конфликта внутри команды.
2. Решение SA/PO при невозможности согласовать внутри команды.
3. Обращение к ментору/преподавателю для внешнего решения.
