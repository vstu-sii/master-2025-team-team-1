# Stakeholder Map

## 1. Primary Stakeholders (Основные заинтересованные стороны)

* **HR-специалисты (рекрутеры)** — проводят массовый отбор, основная целевая аудитория.  
* **Руководители команд / менеджеры по найму** — принимают финальные решения по найму, нуждаются в качественной подготовке к интервью.  
* **Владелец компании (куда ведётся найм)** — принимает стратегические решения, определяет бюджет и KPI найма.  

## 2. Secondary Stakeholders (Вторичные заинтересованные стороны)

* **Ментор** — наблюдает и оценивает ход проекта и предоставляет рекомендации.  
* **Внешний эксперт по HR Tech** — консультации по лучшим практикам и рынку HR-технологий.

## 5. Communication Strategy (Стратегия взаимодействия)

| Stakeholder                         | Communication Strategy (стратегия)                                                      |
| ----------------------------------- | --------------------------------------------------------------------------------------- |
| HR-специалисты (рекрутеры)          | Вовлекать (регулярные интервью, тестирование прототипов, сбор обратной связи)           |
| Руководители команд / менеджеры по найму | Консультироваться (сбор требований, согласование критериев отбора, обратная связь)  |
| Владелец компании                    | Управлять (еженедельные апдейты, согласование бюджета и KPI, стратегические решения)   |
| Ментор                               | Консультироваться (проверка артефактов, советы, письменное подтверждение/ОК)            |
| Внешний эксперт по HR Tech           | Информировать (демо, периодические апдейты, запросы за консультацией при необходимости) |
